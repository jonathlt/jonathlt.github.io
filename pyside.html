Installation - Not Quite Platform Independent!

Windows

  • Download python: (Use x86 2.7 for now)
  • Get Pip https://bootstrap.pypa.io/get-pip.py (download script)
  • c:\python27\python get-pip.py
  • Add c:\python27\scripts to path
  • run pip install virtualenv
  • mkdir pysidetest
  • virtualenv pysidetest
  • pysidetest\scripts\activate
  • pip install PySide

Linux

  • Get Pip https://bootstrap.pypa.io/get-pip.py (download script)
  • python get-pip.py
  • run pip install virtualenv
  • mkdir pysidetest
  • virtualenv pysidetest
  • source pysidetest\bin\activate
  • pip install PySide

OSX

  • Get Pip https://bootstrap.pypa.io/get-pip.py (download script)
  • python get-pip.py
  • run pip install virtualenv
  • mkdir pysidetest
  • virtualenv pysidetest
  • source pysidetest\bin\activate
  • Download wheel pyside package
  • pip install xx.whl

First Program

						
							#!/usr/bin/python
# -*- coding: utf-8 -*-
import sys
from PySide import QtGui,QtCore

class Hello(QtGui.QMainWindow):

    def __init__(self):
        super(Hello, self).__init__()
        self.initUI()

    def initUI(self):

        grid = QtGui.QGridLayout()

        label_hello = QtGui.QLabel("Hello")
        label_hello.setAlignment(QtCore.Qt.AlignCenter)
        grid.addWidget(label_hello,0,0)

        centralWidget = QtGui.QWidget()
        centralWidget.setLayout(grid)
        self.setCentralWidget(centralWidget)

        self.setFixedSize(150,100)

        self.move(300, 150)
        self.setWindowTitle('Hello GUI')
        self.show()


def main():

    app = QtGui.QApplication(sys.argv)
    ex = Hello()
    sys.exit(app.exec_())


if __name__ == '__main__':
    main()
